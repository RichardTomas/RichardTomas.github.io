# HW-Wireframe
Easier Assignment - Building a Wireframe

## Description
A web design skeleton using HTML/CSS.

## Requirements
index.html for HTML
style.css for CSS

## Specifications
HTML:
  - Header
  - Nav
  - Aside
  - Section
  - Article (With a Heading)
  - Headers
  - Paragraph
  
CSS: 
  - Body
  - Header, nav, and footer
  - Section
  - Aside
  - Article
  - .clear
  - Your text elements: h1, p, etc.
